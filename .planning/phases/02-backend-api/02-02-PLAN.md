  <action>
Create `backend/main.py` with:

1. **Lifespan** (asynccontextmanager pattern):
   - Import `load_model_and_data` from `backend.inference`.
   - Call `load_model_and_data()` to get the state dict.
   - Store on `app.state`: `activation_model`, `layer_names`, `test_images`, `test_labels`.
   - Yield, then cleanup (no-op).

2. **App creation**:
   - `app = FastAPI(title="MNIST CNN Visualizer API", lifespan=lifespan)`
   - Add CORSMiddleware: `allow_origins=["http://localhost:3000"]`, `allow_methods=["*"]`, `allow_headers=["*"]`.

3. **GET /api/health** endpoint:
   - Returns `{"status": "ok", "model_loaded": True, "test_images": len(app.state.test_images)}`.
   - Quick liveness check.

4. **GET /api/random-predict** endpoint (the core endpoint per locked decisions):
   - Pick random index from test_images using `random.randint(0, len-1)`.
   - Get image `test_images[idx:idx+1]` (keeps batch dim) and label `int(test_labels[idx])`.
   - Call `run_inference(image, activation_model, layer_names)` from inference module.
   - Call `image_to_base64(image[0, :, :, 0])` for the input image.
   - Return JSON response (per locked decision -- single call returns everything):
     ```json
     {
       "image": "<base64 PNG of input image>",
       "true_label": 7,
       "prediction": 7,
       "confidence": [{"digit": 0, "probability": 0.001}, ...],
       "activations": [{"layer_name": "conv2d_1", "shape": [26, 26, 8], "maps": ["base64...", ...]}, ...]
     }
     ```
   - This fulfills the locked decision: "Backend serves MNIST images via /random-image endpoint" -- combined with inference results in one call per the other locked decision "Single API call returns everything."

5. **`if __name__ == "__main__"` block**:
   - Run uvicorn: `uvicorn.run(app, host="0.0.0.0", port=8000)`.

Imports needed (all in conda ml env): `contextlib.asynccontextmanager`, `random`, `fastapi.FastAPI`, `fastapi.Request`, `fastapi.middleware.cors.CORSMiddleware`, `uvicorn`, and from `backend.inference`: `load_model_and_data`, `run_inference`, `image_to_base64`.

Do NOT use `@app.on_event("startup")` -- it is deprecated. Use lifespan.
