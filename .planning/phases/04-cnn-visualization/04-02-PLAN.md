Create `frontend/app/components/ConfidenceChart.js` with "use client" directive.

Props: `confidence` (array of 10 `{digit, probability}` objects), `prediction` (integer 0-9).

Implementation:
- Import `{ BarChart, Bar, XAxis, YAxis, Cell, LabelList, ResponsiveContainer }` from "recharts"
- Transform confidence data: map each entry to `{ digit: c.digit.toString(), probability: c.probability * 100 }` for percentage display
- Render inside the existing card pattern (`bg-white rounded-xl shadow-sm border border-gray-200 p-6`) with heading "Confidence"
- Use `ResponsiveContainer width="100%" height={280}` wrapping a `BarChart` with `data={data}` and `margin={{ top: 20, right: 10, left: 10, bottom: 5 }}`
- XAxis: `dataKey="digit"`, tick style `{ fontSize: 13, fill: "#6b7280" }`
- YAxis: `domain={[0, 100]}`, `hide` (percentages on bars are sufficient)
- Bar: `dataKey="probability"`, `radius={[4, 4, 0, 0]}` for rounded top corners
- Per-bar coloring with `Cell`: predicted digit gets `fill="#2563eb"` (accent-600), others get `fill="#e5e7eb"` (gray-200). Compare `parseInt(entry.digit) === prediction`.
- LabelList: `dataKey="probability"`, `position="top"`, `formatter={(v) => v.toFixed(1) + "%"}`, style `{ fontSize: 11, fill: "#6b7280" }`
- If `confidence` is null/undefined, render the card with a gray placeholder message "Select a digit to see confidence scores"
